# Contributing

Contributions are welcome. Please open an issue first to discuss what you'd like to change.

## Development

```bash
# Install dependencies
npm install

# Start playground
npm run dev

# Build
npm run build

# Run tests
npm run test
```

The playground at `playground/` is a minimal Nuxt app with the module loaded locally. Use it to test changes.

## Pull requests

- Keep changes focused — one thing per PR
- If you're fixing a bug, include a description of how to reproduce it
- If you're adding a feature, open an issue first to check if it fits the scope
