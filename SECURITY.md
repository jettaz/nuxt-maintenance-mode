# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report them directly via [jettaz.com](https://jettaz.com). Include a description of the issue and steps to reproduce it. You can expect a response within 5 business days.

If the vulnerability is accepted, a patched version will be released as soon as possible. You will be credited in the release notes unless you prefer to remain anonymous.
